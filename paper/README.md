arXiv accepts **English** manuscripts only — submit `en/`; the `zh/` version is
for domestic preprint/blog distribution.
arXiv 仅收英文稿——提交 `en/`；中文稿用于国内预印本或博客分发。

1. Bundle: `en/main.tex` + `en/fig1_complementarity.png` +
   `en/fig2_delegation.png` (flat zip, no folders).
2. License: **CC BY 4.0** recommended.
3. Primary category: **cs.MA** (Multiagent Systems); cross-list: **cs.CR**
   (Cryptography and Security), **cs.CY** (Computers and Society).
4. Abstract field: paste the abstract text (plain text).
5. Standard packages only (`geometry`, `booktabs`, `graphicx`, `amsmath`,
   `amssymb`, `hyperref`, `xurl`, `caption`, `enumitem`, `microtype`, `xcolor`,

## Provenance | 来源与可复核性

- Figures — schematic by design; generated by `scripts/plot_figures.py`; **no
  empirical series plotted**; both captions say so.
- Case-study claims (§5) — architectural, sourced to primary design literature
  (Ouroboros, EUTXO, Hydra, Kachina, reward-sharing, CIP-1694); no market data
  is used as evidence.
- All 27 references are real, primary-source, and cited in context; numbering
  follows first appearance in the text. The agent-frontier anchors are current:
  AP2 (Google, Sept 2025), x402 (Coinbase/Cloudflare, Sept 2025), ERC-4337
  account abstraction (Buterin et al., 2021).
- The disconfirmation set (§7) deliberately exposes the case system itself to
  refutation.
- 所有主张均可追溯至所引一手文献；个案为架构性分析，不以市场数据为证据；§7 写明
  证伪条件。
