# constraint-layer

A theoretical argument that autonomous agents and deterministic ledgers are
complements: an agent can decide but cannot bind; a proof-bearing, ownerless
ledger can bind but cannot decide.
约束层——论证自主智能体与确定性账本互为补物：智能体能决策却不能绑定；自携证明、
无所有者的账本能绑定却不能决策。

## Thesis | 论点

A generative agent can decide but cannot bind; a deterministic, proof-bearing,
ownerless ledger can bind but cannot decide. They are complements. The argument
runs as a layered chain:
生成式智能体能决策却不能绑定；确定性、自携证明、无所有者的账本能绑定却不能决策。
二者互补。论证为分层链条：

| § | EN | ZH |
|---|----|----|
| §2 | the six things an agent structurally lacks (none fixed by scale) | 智能体结构性缺失的六项（皆非规模可补） |
| §3 | the ledger is the deterministic complement; its four functions | 账本是确定性互补物；其四项功能 |
| §4 | five requirements R1–R5 for an agent-grade constraint layer | 代理级约束层的五项要求 R1–R5 |
| §5 | Cardano read against R1–R5 (architecture, not market) | 以 Cardano 对照五要求（架构而非市场） |
| §6 | the shape of agent commerce (bounded delegation) | 智能体商业的形态（受界委托） |
| §7 | open problems and what would disconfirm the thesis | 开放问题与证伪条件 |
| §8 | coda: who writes the rule | 结语：谁来书写规则 |
